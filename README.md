# Financial Timeline OS V39

단일 HTML 기반의 금융 리서치 대시보드입니다. 여러 종목과 벤치마크를 비교하고, 이벤트 로그를 남기고, 차트 위에서 가이드라인과 알람 기준을 함께 관리할 수 있습니다.

## 주요 기능

- 종목 검색 후 비교 리스트에 추가
- 글로벌 벤치마크 관리 및 메모 저장
- 라인 차트 / 단일 종목 전용 캔들 차트 전환
- 단일 종목 전용 거래량 오버레이 표시
- 가로선 수동 입력 추가
- 차트 위 마우스 드로잉으로 가로선 추가
- 가로선별 색상과 메모 저장/수정/삭제
- 종목별 다중 변동성 알람 설정
- 이벤트 카드 추가, 수정, 복제, 삭제
- 통합 검색에서 종목 / 이벤트 / 벤치마크 메모 검색
- 현재 허브 상태 Snapshot 저장 및 재로딩
- PNG 내보내기

## 화면 구성

### 좌측 패널

- `Comparison Hub`
  - 종목 검색, 비교 종목 추가
  - 벤치마크 추가/삭제
  - 차트 모드 전환
  - 거래량 오버레이 토글
  - 가로선 값/색상/메모 입력
  - 차트 드로잉 모드 토글
  - 기간 선택 및 강제 동기화
- `Insight Logger`
  - 이벤트 유형, 날짜, 제목, 설명 입력
- `Volatility Alert Hub`
  - 종목별로 여러 개의 임계치 알람 설정 가능

### 중앙 리포트

- 선택 종목 비교 차트
- 이벤트 플래그와 툴팁
- 이벤트 카드 리스트
- 카드별 `Edit`, `Copy`, `Delete`

### 우측 패널

- 통합 검색 결과
- Asset Database
- Global Event Feed
- Snapshot 목록
- OS V39 / Snapshot 액션

## 차트 동작

### 차트 모드

- `Line`: 기본 비교 모드
- `Candle`: 단일 종목일 때만 활성

### 거래량 오버레이

- 단일 종목일 때만 `Volume Overlay` 버튼 사용 가능
- OHLC 응답에 `volume` 값이 있으면 차트 하단에 거래량 바가 함께 표시됨

### 가로선

- 입력 방식
  - 값 입력 후 `+ Line`
- 드로잉 방식
  - `Draw On Chart`를 `Armed`로 전환
  - 차트 위에서 마우스를 움직이면 프리뷰 선 표시
  - 원하는 위치 클릭 시 가로선 추가
  - 그래프 영역을 벗어나면 프리뷰 선은 즉시 사라짐
- 각 가로선은 다음 속성을 가짐
  - 값
  - 색상
  - 메모
  - 현재 차트 모드 기준 타입(`Price` 또는 `Return %`)

## 이벤트 카드

- 카드 클릭 시 해당 이벤트를 차트에서 하이라이트
- `Edit`
  - 날짜, 유형, 제목, 설명 수정
  - 날짜는 현재 로드된 가격 이력에 존재하는 거래일만 허용
- `Copy`
  - 동일 내용을 새 이벤트로 복제
- `Delete`
  - 개별 이벤트 ID 기준 삭제

## 알람 허브

- 동일 종목에 여러 개의 알람 임계치 추가 가능
- 종목별 알람은 개별 수정/삭제 가능
- 차트에서 세로 경고선으로 시각화 가능

## 데이터 및 저장

브라우저 `localStorage`를 사용합니다.

- `rd_v39_db`: 종목 메타데이터와 이벤트
- `rd_v39_combined`: 저장된 Snapshot 세트
- `rd_v39_bench`: 사용자 벤치마크 목록

## API 기대 형식

앱은 `CF_WORKER_BASE` 기준으로 다음 엔드포인트를 사용합니다.

- `/search?q=...`
- `/quote?ticker=...`
- `/history?ticker=...&start=...&end=...`

`/history`는 최소 `date`와 `price` 또는 `close`를 포함해야 하며, 아래 필드를 지원합니다.

```json
{
  "date": "2026-03-28",
  "open": 100,
  "high": 110,
  "low": 95,
  "close": 108,
  "volume": 1234567
}
```

- `open/high/low/close`가 있으면 캔들 차트 렌더링 가능
- `volume`이 있으면 거래량 오버레이 렌더링 가능

## 실행 방법

정적 파일이라 별도 빌드는 필요 없습니다.

1. 저장소를 클론합니다.
2. `index.html`을 브라우저에서 열거나 간단한 정적 서버로 제공합니다.
3. `CF_WORKER_BASE`가 접근 가능한 API를 가리키는지 확인합니다.

예시:

```bash
python3 -m http.server 8000
```
